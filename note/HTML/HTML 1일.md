[[HTML]]

1. HTML 기본 구성
========================================
- <html> : 문서의 시작과 끝을 알림
- <head> : 문서의 정보(메타데이터, 제목 등)를 정의 (화면에 안 보임)
- <body> : 실제 브라우저 화면에 출력되는 내용을 담음

1. 헤드 영역 태그
========================================
- <title> : 웹 브라우저 탭에 표시될 제목 설정
- <meta>  : 문서의 정보 설정

3. 텍스트 관련 태그
========================================
(1) 제목 달기
   - <h1> ~ <h6> : 숫자가 작을수록 글자가 크고 중요함 (h1이 가장 큼)
   - 툴팁 속성 : <h1 title="설명"> (마우스를 올리면 설명이 뜸)

(2) 본문 및 서식
   - <p>   : 문단 나누기 (위아래 여백 생김)
   - <br>  : 줄 바꿈 (닫는 태그 없음)
   - <hr>  : 수평선 긋기
   - <pre> : 입력한 포맷(공백, 엔터) 그대로 출력

4. 구조 및 레이아웃
========================================
- <div> : 구역을 나누는 블록 태그 (CSS 스타일링이나 레이아웃 잡을 때 주로 사용)

5. 멀티미디어
========================================
- <img> : 이미지 삽입
   * 필수 속성: src(경로), alt(이미지 설명/대체 텍스트)

6. 리스트 (목록) 만들기
========================================
- <ul> : 순서가 없는 목록 (점으로 표시)
- <ol> : 순서가 있는 목록 (1, 2, 3... 숫자 표시) 속성에 따라 로마숫자, 알파벳
- <li> : 목록의 각 항목 (ul/ol 안에 위치)

7. 표 (Table) 만들기
========================================
(1) 기본 태그
   - <table> : 표 생성
   - <tr>    : 행(Table Row, 가로줄) 생성
   - <th>    : 제목 셀 (Table head, 가운데 정렬, 굵게)
   - <td>    : 일반 데이터 셀

(2) 주요 속성 (병합 및 크기)
   - colspan : 가로로 셀 합치기 (열 병합)
   - rowspan : 세로로 셀 합치기 (행 병합)
   - width   : 표 또는 셀의 너비
   - height  : 표 또는 셀의 높이
