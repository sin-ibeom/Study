 1. 링크 만들기 : <a> 태그
========================================
- 기능: 텍스트나 이미지를 클릭하면 지정된 주소로 이동
- 기본 문법:
  <a href="이동할 주소" target="창 여는 방식">클릭할 글씨</a>

- 주요 속성:
  * href : 이동할 경로 또는 주소
  * target="_blank": 새 탭에서 열기

1. 페이지 내부 이동
========================================
- 긴 페이지에서 특정 위치로 스크롤 이동할 때 사용
- 원리: 도착할 곳에 id를 주고, 링크에는 #id를 적음

  <p id="top">여기가 맨 위입니다</p>

  <a href="#top">맨 위로 이동</a>

3. URL 스킴 (Scheme)
========================================
- 웹 브라우저가 아닌 특정 어플리케이션을 실행하는 주소 체계
- 예시:
  * <a href="tel:010-1234-5678">전화 걸기</a> (모바일에서 전화 앱 실행)
  * <a href="mailto:abc@naver.com">메일 보내기</a> (메일 앱 실행)

4. 주소 경로 체계 (★중요)
========================================
(1) 절대 주소 (Absolute Path)
   - 형식 1 (완전한 주소): https://, http://, // 로 시작
     * 예: https://www.naver.com
   - 형식 2 (루트 상대 경로): / 로 시작
     * 도메인(IP:Port)은 유지하되, 그 뒤의 경로를 싹 지우고 시작함.
     * 예: /login (현재 사이트의 최상위 폴더 아래 login으로 이동)

(2) 상대 주소 (Relative Path)
   - 절대 주소 형식이 아닌 모든 경우 (그냥 파일 명이나 폴더 명으로 시작)
   - 특징: 현재 주소창의 맨 마지막 슬래시(/) 뒤를 변경함.
   
   <특수 기호>
   * ./ (점 1개) : 현재 폴더 (생략 가능)
     * 예: ./image.jpg == image.jpg
   * ../ (점 2개) : 부모 폴더 (상위 폴더로 이동)
     * 예: ../css/style.css (한 단계 위로 올라가서 css 폴더 찾기)
