# 선택자

## 전체 선택자 : *
- 현재 페이지에 있는 모든 tag를 선택한다

## 태그 선택자
```
h1 { 프로퍼티 : 값 }
```
- 그냥 태그 명을 적는다
- 모든 태그에 적용된다

## 다중 선택자 : 여러 선택자와 같은 내용으로 한번에 적용
- 선택자들을 ,로 나열하기
```
선택자1, 선택자2 { ... }
```

## class 선택자 : 클래스 명 앞에 .을 찍는다

```
<style>
	.c1 { color : red; }
</style>

<h1 class ="c1 c2">내용</h1>
```
- class 속성은 모든 태그에 적용할 수 있다
- 중복을 허용한다
- 띄어쓰기를 이용하여 여러개의 클래스를 가질 수 있다.

## id 선택자 : id 앞에 '#'을 찍는다
```
<style>
	#content { color : red; }
</style>

<h1 id="content">내용</h1>
```
- id속성은 모든 태그에 적용할 수 있다.
- 중복은 가능하면 피한다
- 한 페이지에 딱 하나의 id만 놓는 것이 기본
- 값은 하나만 가질 수 있다.
# 자식 선택자
- 바로 아래 자식 요소만 선택( 1촌 관계 , 직계)
```
<style>
	div > strong { color : red; }
</style>

<div>
	<strong>div 안에 strong</strong>
</div>
```

## 자손 선택자
- 자식의 자식 등 하위에 있는 자손 중에서 선택된다.
```
<style>
	div strong { color : red; }
</style>

<div>
	<a href=''>
	<strong>div > a > strong</strong>
	</a>
</div>
```


## 속성 선택자
- [속성명] : 해당 속성이 있는 모든 것
- [속성명=값] : 속성이 존재하고 값도 일치하는 모든 것
- [속성명^=값] : 속성이 존재하고 값으로 시작하는 모든 것
- [속성명$=값] : 속성이 존재하고 값으로 끝나는 모든 것

## 가상 선택자
```
선택자:가상 선택자 { ... }
```

- hover : 마우스 올리면 적용
- active : 마우스 클릭 시 적용
- focus : 선택되면 적용
- nth-child(숫자) : 선택자 중에서 몇 번째 자식을 지정할 때 사용
	- 2n : 짝수
	- 2n-1 : 홀수
- after : 선택자 뒤에 content 추가 및 content의 CSS 추가
	- javascript로 변경할 수 없다.
- ```
  h3:after {
	  content: " AFTER";
	  color : black;
	  ;}
  ```

- checked : checkbox, radiio 등이 선택 됐을 때 적용 됨

## 부정 선택자
- not (선택자) 
	- 선택자:not(선택자2)  : 선택자 중에 선택자2를 제외하고 선택

## 형제 선택자
- + : 바로 다음 형제를 선택
```
input[type=checkbox]:checked + span {
	text=decoration: line-through;
}
```

## 선택자 우선 순위
- 구체적일 수록 우선 순위가 높다.
- 동일한 우선순위인 경우 나중에 나온 것으로 덮어쓴다.
```
1. !important  
2. style 속성  
3. #id 선택자  
4. 속성 선택자 - 클래스 선택자  
5. 태그 선택자  
6. 전체 선택자  
7. 브라우저 기본 값
```




# 단위

## 색상
- '#'으로 시작하는 16진수 두자리 씩 RGB, RGBA 표현
	- #00FFAA 처럼 두글자가 반복 될 때 : #0FA 로 축약 가능

- RGB() , RGBA()로 0~255까지 RGBA를 표현

## 크기
- px : 화면의 점 하나 크기 ( 절대 )
	- 기기마다 크기가 다를 수 있다.
- % : 현재 값에 대한 상대 크기
- em : 현재 값에 대한 상대 크기( 보통 글씨의 단위로 씀 )
	- rem : root em, 최상단의 크기
- vh : 화면의 높이를 1/100로 쪼갠 단위
- vw : 화면의 너비를 1/100로 쪼갠 단위
- vmin : 너비나 높이 중 작은 것을 1/100로 쪼갠 단위
- vmax : 너비나 높이 중 큰 것을 1/100로 쪼갠 단위
